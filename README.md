## Instalação

```sh
yarn global add vovo-juju-cli
```

## Bonus

### Fala da Vovó Juju toda vez que abrir o terminal

1. Edite `~/.bashrc` ou `~/.zshrc`;
1. Coloque `vovo-juju` no fim do arquivo;
1. Reabra o terminal.
